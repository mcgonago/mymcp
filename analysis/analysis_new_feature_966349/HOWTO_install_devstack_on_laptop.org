* TL;DR

This page describes how to install devstack following the docs at [[https://docs.openstack.org/devstack/latest/][docs.openstack.org/devstack/latest/]]

You have a few choices:
- get a PSI account and install devstack in your psi instance
- get *Virtual Machine Manager* (virt-manager) and install devstack under Ubuntu 22.04 VM on your laptop

This page describes using virt-manager on your laptop

* Point your browser at your new Horizon controlled devstack
After completing [[https://spaces.redhat.com/display/OSP/Install+devstack+on+laptop#Installdevstackonlaptop-cloneandrun(tox)horizonondevstack][clone and run (tox) horizon on devstack]] - point your browser at your VM, and login as admin
