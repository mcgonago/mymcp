* TL;DR

This page describes how to install devstack following the docs at [[https://docs.openstack.org/devstack/latest/][docs.openstack.org/devstack/latest/]]

You have a few choices: 
- get a PSI account and install devstack in your psi instance
- get *Virtual Machine Manager* (virt-manager) and install devstack under Ubuntu 22.04 VM on your laptop

This page describes using virt-manager on psi

The steps for installing devstack on your laptop is at [[file:HOWTO_install_devstack_on_laptop.org][Howto Install Devstack on your Laptop]]

This page describes how to create a PSI instance that supports Horizon ui-pytest/ui-integration-pytest devstack/tox

I already have documented how to install *Fedora40* and *RHEL* on PSI (to support install_yamls) at: 
- [[https://spaces.redhat.com/spaces/OSP/pages/527410011/Install+Fedora40+on+PSI][Install Fedora40 on PSI]]
- [[https://spaces.redhat.com/spaces/~omcgonag/pages/627081543/horizon_dfg_ui_howto_article_install_rhel_on_psi][Install RHEL on PSI]]

# Where the keystone endpoint is
auth_url=http://localhost/identity/v3
#+END_SRC

* Perform basic steps to add stack user
We do this as root first
:  ssh -i ~/.ssh/id_rsa_omcgonag_key_pair_devstack_2 ubuntu@10.0.148.47
:  sudo -i

Do a few updates
:  sudo apt update
:  sudo apt upgrade
:  sudo apt install emacs-nox

Add user and authorized keys from ubuntu

:  sudo useradd -s /bin/bash -d /opt/stack -m stack
:  sudo chmod +x /opt/stack
:  echo "stack ALL=(ALL) NOPASSWD:  ALL" | sudo tee /etc/sudoers.d/stack
:  sudo chown root: root /etc/sudoers.d/stack
:  sudo chmod 0660 /etc/sudoers.d/stack

:  if [ ! -e /opt/stack/.ssh/authorized_keys ]; then
:        sudo mkdir -p /opt/stack/.ssh
:        sudo chmod 0700 /opt/stack/.ssh
:        sudo cp /home/ubuntu/.ssh/authorized_keys /opt/stack/.ssh/authorized_keys
:        sudo chown -R stack:  /opt/stack/.ssh
:        sudo chmod 0600 /opt/stack/.ssh/authorized_keys
:  fi

A bit about *local.conf*

#+BEGIN_SRC bash   # local.conf
[[local|localrc]]
ADMIN_PASSWORD=xxxxxx
DATABASE_PASSWORD=$ADMIN_PASSWORD
RABBIT_PASSWORD=$ADMIN_PASSWORD
SERVICE_PASSWORD=$ADMIN_PASSWORD
This is the minimum required config to get started with DevStack.
#+END_SRC

NOTE: 
- There is a sample local.conf file under the samples directory in the devstack repository.
- Only use alphanumeric characters in your passwords, as some services fail to work when using special characters.

:  git clone https://opendev.org/openstack/devstack
:  cd devstack

Start the install
:  ./stack.sh

This will take 15 - 30 minutes, largely depending on the speed of your internet connection.
Many git trees and packages will be installed during this process.
