** Use local_settings.d to toggle between angular and python code
We can configure the *ANGULAR_FEATURES* toggle between angular vs. python by
- updating horizon/openstack_dashboard/defaults.py
- local_settings.d: renaming *_11_toggle_angular_features.example* to *_11_toggle_angular_features.example.py*

** defaults.py
For example - to inspect the differences for the *key pairs* panel
- toggle *key_pairs_panel* between *True* and *False*
- any change to this file triggers a restart of the Horizon dashboard
- *CAREFUL:* if not careful, this file could get pushed/changed during a commit (local_settings.d changes do not get pushed)

: X
: iyaml horizon/openstack_dashboard/defaults.py
:
: # Dictionary of currently available angular features
: ANGULAR_FEATURES = {
:     'images_panel': True,
:     'key_pairs_panel': True,
:     'flavors_panel': False,
:     'domains_panel': False,
:     'users_panel': False,
:     'groups_panel': False,
:     'roles_panel': True
: }
